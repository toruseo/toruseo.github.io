About

オープンソース交通流シミュレータUXsimのポータルサイトです． 最新版のシミュレータにはGitHubからアクセスできます． 書籍『マクロ交通流シミュレーション：数学的基礎理論とPythonによる実装』に掲載したバージョンも記録としてGitHubに保存しています．

GitHub リポジトリ

UXsimはPython製のオープンソース・フリーなマクロ・メソ交通流シミュレータです． これは以下のモデルを組み合わせてネットワークの動的交通流を計算するものです：

  • Newellの単純追従モデル（Xモデル）
  • Lagrange版Incremental Node Model
  • Dynamic User Optimum型経路選択モデル（慣性付き）

このサイトは，書籍に記載したURLを出来るだけ長期間にわたって有効なものとするために用意したポータルサイトです （例えばGithubのURLを書籍に記載した場合，サービスの変更・終了とともにアクセスできなくなってしまうため）． シミュレータの詳細については以下の専門サイト・資料をご覧ください．
