UXsim: Python製のオープンソース交通流シミュレータ

About

オープンソース交通流シミュレータUXsimのページです． 最新版のシミュレータにはGithubからアクセスできます． 書籍『マクロ交通流シミュレーション...』に掲載したバージョンも記録としてGithubに保存しています．

UXsimはPython製のオープンソース・フリーなマクロ・メソ交通流シミュレータです． これは以下のモデルを組み合わせてネットワークの動的交通流を計算するものです：

  • Newellの単純追従モデル（Xモデル）
  • Lagrange版Incremental Node Model
  • Dynamic User Optimum型経路選択モデル（慣性付き）
